# LAW_Assistant 智能法律咨询 Agent

## 📖 项目简介
LAW_Assistant 是一个基于 Agentic AI 架构搭建的智能法律决策支持助手。该 Agent 旨在为普通用户提供低门槛、高准确率的专业法律咨询服务。底层结合了检索增强生成（RAG）技术与结构化工作流，能够精准解析复杂案情（如婚姻财产、合同违约、侵权索赔等），并输出具备实务指导价值的法律方案。

## ✨ 核心特性 (Core Features)
- **逻辑分层输出**：严格按照“一般规则 -> 例外与冲突情形 -> 适用结论与界限”的三段论结构输出，降低普通用户的认知门槛。
- **多步推理工作流 (Workflow)**：通过内置的 `LAW_Assistant` 等工作流，对用户输入的非结构化案情进行要素提取、法条匹配和风险预判，避免模型发散。
- **合规与风控隔离**：内置情绪安抚与风险阻断机制，引导用户采用合法合规的途径（如证据保全、诉讼途径）维权，有效规避“私力救济”带来的法律反噬风险。
- **动态知识检索 (RAG)**：深度外挂《中华人民共和国民法典》及相关司法解释知识库，确保引用的法条和审判口径具备时效性与准确性。

## 🏗️ 架构设计 (Architecture)
本项目主要包含以下核心模块：
1. **Persona & Prompt**：定义 Agent 的专业律师助手人设，强制规范输出的逻辑骨架与合规界限。
2. **Workflows**：处理复杂的法律逻辑分支（如判断是否构成“无权处分”、“正当防卫”等），保障大规模调用时输出的稳定性与一致性。
3. **Knowledge Base**：结构化存储的法律法规文本、指导性案例及实务审判口径。
4. **Plugins**：预留联网搜索等插件，用于实时获取最新法律法规或地方性政策变动。

## 🚀 部署与分发 (Deployment)
当前 Agent 具备跨平台分发能力，适合推向普通用户使用：
- **Web SDK**：可通过独立网页嵌入，结合前端 UI 提供开箱即用的法律咨询交互窗口。
- **社交与办公生态**：支持一键配置发布至微信公众平台、飞书等主流应用。

### 接入指引示例 (Web 独立页)
1. 在 Agent 控制台的“发布”页面获取 Web SDK 凭证 (App ID)。
2. 将初始化代码片段嵌入至目标独立站点的 HTML 文件中。
3. 建议在前端配合配置“快捷引导问”（如“房东不退押金怎么办？”）与全局免责声明。
