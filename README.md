# ⚖️ 法律查询助手 (Legal Retrieval Agent)

本项目是一个专为法律从业者（律师、法务、法官、法学生）打造的高精度法律检索与释义 Agent。基于大语言模型与深度优化的 RAG（检索增强生成）技术，本项目以《中华人民共和国民法典》为底层知识库，致力于解决传统关键词检索的“语义鸿沟”以及通用大模型的“法律幻觉”痛点。

## ✨ 核心特性

- 🎯 **语义穿透召回**：支持从复杂的“隐性事实”与“多重法律关系”中，精准提取争议焦点并映射至对应法条。
- 🧩 **结构化法理解析**：摒弃冗余口语，严格遵循“法条引用 -> 法理释明 -> 实务指引”三段式输出，自动拆解法定构成要件与优先适用规则。
- 🛡️ **严守安全红线**：内置防零召回熔断机制。检索为空时强制阻断生成，彻底杜绝大模型凭空编造法条的合规风险。
- 📊 **专业基准测试**：自带高质量、多梯度的《民法典》专属评测集（涵盖显性事实、隐性事实、逻辑推理与对抗样本），支持自动化评测召回率与精确率。
